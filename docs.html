Quick Start Guide

Get up and running in 5 minutes

1

Download and Extract

Download the latest version from our download page and extract the ZIP file to a folder of your choice.

Double-click the executable file to launch DriveBackup Pro. The application will open with a modern, intuitive interface.

3

Configuration Guide

Learn how to configure Google API credentials and customize settings

Google API Setup

To use DriveBackup Pro, you need to set up Google API credentials:

1. Create a Google Cloud Project

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Google Drive API

2. Create OAuth2 Credentials

  1. Navigate to "APIs & Services" > "Credentials"
  2. Click "Create Credentials" > "OAuth 2.0 Client IDs"
  3. Choose "Desktop application" as the application type
  4. Download the credentials file as credentials.json

3. Configure Redirect URIs

Add these redirect URIs to your OAuth2 client:

http://localhost:8080/
http://127.0.0.1:8080/

Core Functions

authenticate()

Authenticates with Google Drive using OAuth2

def authenticate():
    """Authenticate with Google Drive API"""
    flow = InstalledAppFlow.from_client_secrets_file(
        'credentials.json', SCOPES)
    credentials = flow.run_local_server(port=8080)
    return build('drive', 'v3', credentials=credentials)

download_folder(folder_id, local_path)

Downloads all files and subfolders from a Google Drive folder

def download_folder(folder_id, local_path):
    """Download entire folder structure"""
    files = service.files().list(
        q=f"'{folder_id}' in parents",
        fields="files(id,name,mimeType,size)").execute()

download_file(file_id, local_path)

Downloads a single file with format conversion support

def download_file(file_id, local_path):
    """Download single file with conversion"""
    if is_google_workspace_file(mime_type):

    else:
        return download_regular_file(file_id, local_path)

Frequently Asked Questions

Common questions and troubleshooting solutions

How do I set up Google API credentials?

Follow our Configuration Guide for step-by-step instructions. You'll need to create a Google Cloud project, enable the Drive API, and create OAuth2 credentials.

What file formats are supported for conversion?

We support converting Google Docs to Word (.docx), Google Sheets to Excel (.xlsx), and Google Slides to PowerPoint (.pptx). PDF export is also available for all formats.

How does incremental backup work?

Incremental backup compares file sizes and modification dates to skip files that haven't changed. This saves time and bandwidth for subsequent backups.

Can I backup shared drives?

Yes! DriveBackup Pro supports backing up all shared drives you have access to. Simply enable the "Shared Drives" option in the backup settings.

What if the backup gets interrupted?

You can resume interrupted backups by running the application again. It will automatically skip files that were already downloaded successfully.

How do I handle large files?

The application handles large files automatically with progress tracking and retry logic. For very large files, ensure you have a stable internet connection.
